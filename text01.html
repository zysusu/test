下载安装git和首次发布项目到GitHub

2017-6-20

     想必对于所有程序员来说，都希望能打造一个属于自己的网站。尤其是初级程序员们，都迫切的希望看到一些开发成果。奈何自己能力尚欠缺，各种相关的博客，资料也是“五花八门”。本文则分享了几个博主如何从只会写写简陋网页的小白，到创建自己的博客并发布到GitHub上的优秀博客，希望对你们在GitHub上搭建博客能有帮助。

    Step1. github与hexo起步:http://hifor.net/2015/07/01/零基础免费搭建个人博客-hexo-github/

    Step2.Hexo安装与配置：http://www.cnblogs.com/zhcncn/p/4097881.html

    Step3. 创建本地与github项目连接:http://www.cnblogs.com/Qhonhu/p/6137090.html

    通过以上三个博客介绍，你就能成功发布第一个项目到GitHub上了。此后，如果本地项目有修改，要再次上传更新。则会方便很多，主要步骤如下：

    第一步：打开项目文件夹，单击右键选中”Git Bash Here”，然后则可打开git命令行，先输入git init初始化；最好再输入git status查看修改状态；

    第二步：将文件commit到仓库： Git commit -m “注释语句”

    第三步：上传github之前，要先pull一下：Git pull

    第四步：上传代码到github远程仓库： Git push -u

    执行完后，如果没有异常，等待执行完就上传成功了，则可打开github上的网址查看是否是修改后的状态啦。
